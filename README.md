# GS Fresh 예약 관리 (POC)

매장에서 전화/방문 예약을 받은 물품을 모바일·태블릿에서 관리하기 위한 간단한 POC입니다.

## 주요 기능

- **예약 조회**: 전화번호 뒷 4자리로 해당 고객의 예약 내역을 한 번에 조회
- **수령 체크**: 제품별 수령 처리/취소, 고객 단위 일괄 처리 지원
- **미수령 현황**: 전체 미수령 건을 고객별로 그룹핑, 기한 지남 건 우선 노출
- **CSV 내보내기**: 현재 상태를 CSV로 다운로드하여 엑셀에서 재업로드 가능
- **로컬 저장**: 수령 처리 상태는 브라우저 `localStorage`에 저장 (POC용)

## 파일 구성

- `index.html` — 단일 페이지 UI
- `style.css` — 모바일/태블릿 반응형 스타일
- `app.js` — CSV 로드, 조회, 수령 토글, 미수령 렌더링
- `reservations.csv` — 예약 데이터 (30행 샘플)

## 로컬 실행

정적 파일이므로 어떤 정적 서버에서도 동작합니다.

```bash
python3 -m http.server 8080
# http://localhost:8080 접속
```

## 데모 기준일

`app.js` 내의 `TODAY` 상수가 `2026-04-19`로 고정되어 있습니다. 샘플 데이터의 날짜에 맞춰 D-day/기한지남 표시가 의미있게 보이도록 한 것으로, 실운영 시 `new Date()`로 바꾸면 됩니다.
